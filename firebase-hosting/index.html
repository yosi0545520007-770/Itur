Itur Geocoder

This Firebase Hosting deployment provides project info and quick start instructions for the Streamlit-powered geocoding tool.

Run locally

  1. Clone the repository from GitHub.
  2. Create a virtual environment and install dependencies:
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -e ".[dev]"
./scripts/setup.ps1
./scripts/serve.ps1

The Streamlit app will open on http://localhost:8501.

Data privacy

Make sure you configure your Google Maps API key via GOOGLE_MAPS_API_KEY environment variable or .streamlit/secrets.toml.
